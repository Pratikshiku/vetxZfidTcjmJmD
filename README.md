### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, blogger users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Blogger Articles
5. System Announcements
5. Personal Center (Personal Center, My Favorites)


**Blogger User Functions:**
1. Login
2. Registration
3. Homepage
4. Blogger Articles
5. System Announcements
6. Personal Center (Personal Center, My Favorites)



**User Backend Functions:**
1. Login
2. Homepage
3. Personal Center (Change Password, Personal Center)


**Blogger Backend Functions:**
1. Login
2. Registration
3. Homepage
4. Personal Center (Change Password, Personal Center)
5. Blogger Management
6. Blogger Article Management

**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center (Change Password, Personal Center)
4. User Management
5. Blogger Management
6. Article Category Management
7. Blogger Article Management
8. System Management (System Announcements, Carousel Management)
